# vuescroll-ssr-php

> The repo uses vue-ssr ,vuescroll under v8js , Laravel and php environment.

## Get Started.

Assume you have had v8, v8js, php in your computer.

1. Install vender

```bash
   composer  install
```

2. Start php server

```bash
   php artisan serve
```

3. install node_modules

```bash
   npm i
```

4. Generate js files,

```bash
  npm run dev
```

5. visit `http://127.0.0.1:8000/`
